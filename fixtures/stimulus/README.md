# Procedural Stimulus Fixtures

These fixtures describe clean-room renderer-neutral procedural stimulus
contracts. They are derived from public reference behavior and OSF metadata
only; Trevor Hewitt's unlicensed source and hosted assets are not copied into
this repository.

`interference_preview_profile.json` is a low-frequency contract fixture for
full-screen stereo-eye wiring, shader parameter mapping, CPU reference samples,
and display-cadence run-plan checks. It is not a study protocol. Research
protocol consent, session gating, and participant handling are owned by the
external experiment workflow, while adapters should still surface profile
metadata and installer-facing notices.
